# # Medicine and Drug Ordering System

A web-based application designed to streamline the process of browsing, managing, and ordering pharmaceutical products. This project was developed as part of a university module to demonstrate full-stack development and database management.

## ✨ Features
* **User Authentication:** Secure login and registration for customers and pharmacists.
* **Inventory Management:** Real-time tracking of medicine stock levels and expiry dates. (my part)
* **Search & Filter:** Easily find drugs by category, name, or manufacturer.
* **Order Processing:** Seamless checkout and order history tracking.

## 🛠️ Tech Stack
* **Frontend:** (e.g., HTML, CSS, JavaScript / React)
* **Backend:** (e.g., Java Springboot)
* **Database:** (e.g., MySQL)
